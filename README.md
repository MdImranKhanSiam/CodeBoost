# 🚀 CodeBoost

**CodeBoost** is a scalable online judge platform built for competitive programming, technical interview preparation, and real-time coding contests.  

It provides a secure environment for multi-language code execution, automated test case evaluation, and performance-based rankings — ensuring a fair and reliable judging experience for all users.

CodeBoost is designed to support high submission volumes, structured contest management, and extensible architecture for future scalability.

[![Live Website](https://img.shields.io/badge/Live%20Website-CodeBoost-blue?style=for-the-badge)](https://codeboost.online)

---

# 📌 Features

## ⚙️ Code Execution & Automated Judging
- Secure sandboxed code execution
- Multi-language support (C, C++, Python, Java, etc.)
- Time limit and memory limit enforcement
- Public and hidden test case evaluation
- Detailed verdict system:
  - Accepted (AC)
  - Wrong Answer (WA)
  - Time Limit Exceeded (TLE)
  - Memory Limit Exceeded (MLE)
  - Runtime Error (RE)
  - Compilation Error (CE)
- Submission history tracking

---

## 🏆 Contest Management
- Real-time contest hosting
- Scheduled and private contests
- Dynamic leaderboard updates
- Penalty-based ranking system
- Live submission status updates
- Contest participation tracking

---

## 📊 User & Performance System
- Secure authentication & authorization
- Role-based access control (Admin / Problem Setter / User)
- Rating and ranking system
- User profile statistics
- Problem-solving analytics

---

## 🔐 Security & Isolation
- Sandboxed execution environment
- Resource isolation per submission
- Protection against malicious code execution
- Rate limiting and abuse prevention

---

## 🚀 Scalability & Architecture
- Asynchronous submission processing
- Queue-based judging pipeline
- Support for distributed judge workers
- Optimized database indexing
- Horizontal scaling ready
- Caching for performance optimization

---

## 🌐 Real-Time Capabilities
- WebSocket-based live updates
- Instant verdict notifications
- Real-time leaderboard refresh
- Live contest activity tracking

# 🏗️ System Design Goals

- Reliability under high concurrency
